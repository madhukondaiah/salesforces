# Salesforce Project

A Salesforce development project featuring Apex triggers, process automation, and configuration documentation.

## Project Structure

```
salesforces/
├── src/
│   └── DropOfTrigger.txt     # Apex trigger source code
├── docs/
│   └── salesforce-project.docx  # Project documentation
├── screenshots/
│   └── *.png / *.jpg          # UI screenshots and demo images
└── README.md
```

## Features

- Apex Trigger implementation (Drop of Trigger)
- Salesforce declarative configuration
- Project walkthrough documentation with screenshots

## Tech Stack

- Salesforce Apex
- Salesforce Lightning / Classic UI
